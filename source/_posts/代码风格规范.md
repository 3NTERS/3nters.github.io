## Setup.py

将包安装到环境可以极方便调用包内函数。

- `__init__.py`声明文件夹是包

- `pip install -e .`安装

- 包名会自动将最外包含`__init__.py`文件夹作为包

  > 上面项目就是将detector文件夹作为包的内容

```
from setuptools import setup, find_packages
setup(
   name='detector', # 包名
   version='0.1', # 版本号
   packages=find_packages(),
   install_requires=[ # 列出依赖包
       # 'numpy>=1.21.0',
   ],
   description='A detector(screws drill drill_tip) Python package',
   author='xxx',
   author_email='your_email@example.com',
)
```
